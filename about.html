About

Merhaba, ben Mert Bingöl. Yazılım ve siber güvenlik konularına odaklı bir web application developer olarak, özellikle web güvenliği konusunda bilgi birikimim olması için çalışmalar gerçekleştiriyorum. Web uygulamalarının backend tarafında da çalışıyorum. Blogumda, web uygulamalarının güvenliği, geliştirilmesi, test edilmesi ve daha birçok güvenlik konularına değineceğim ve sizlere web güvenliği konusunda bilgi ve ipuçları sunacağım. Ayrıca, siber güvenlik konularının yanı sıra, yazılım geliştirmeyle ilgili konulara da kafa yoracağız. Edindiğim bilgileri sizlere anlatmam gerekiyor çünkü Mehmet abinin dediği gibi, "edindiğim bilgileri paylaşmayı borç bilirim".

Skills

Python ile backend web uygulamalarının yazılması konusunda orta seviyede bilgi ve tecrübem var. Ayrıca, API entegrasyonları gerçekleştirebilir, veritabanı işlemleri yapabilir ve web scriptleri oluşturabilirim. Aynı zamanda Web uygulamalarının güvenliği konusunda da bilgi birikimine sahibim, her geçen gün öğrenmeye devam ediyor ve arada sırada bugbounty yoluyla da bu konularda deneyim kazanmaya çalışıyorum. Bu öğrenme sürecimde edindiğim bilgileri burada sizlerle paylaşıyor olacağım.
